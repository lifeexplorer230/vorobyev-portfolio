# САЙТ 1: https://mafclub.biz/

## КРИТИЧНЫЕ ПРОБЛЕМЫ

### 1. Отсутствует robots.txt
**Где:** https://mafclub.biz/robots.txt
**Проблема:** Файл возвращает 404 ошибку. Поисковые системы не получают инструкции по индексации сайта.
**Как исправить:**
- Создать файл robots.txt в корне сайта
- Минимальное содержимое:
```
User-agent: *
Allow: /
Disallow: /consent.html
Sitemap: https://mafclub.biz/sitemap.xml
```
**Влияние:** Критичное для SEO. Без robots.txt поисковики могут индексировать технические страницы или пропускать важные разделы.

---

### 2. Отсутствует sitemap.xml
**Где:** https://mafclub.biz/sitemap.xml
**Проблема:** Файл возвращает 404 ошибку. Поисковые системы не имеют карты сайта.
**Как исправить:**
- Создать sitemap.xml со всеми страницами:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://mafclub.biz/</loc>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://mafclub.biz/offer.html</loc>
    <changefreq>monthly</changefreq>
    <priority>0.3</priority>
  </url>
  <url>
    <loc>https://mafclub.biz/privacy.html</loc>
    <changefreq>monthly</changefreq>
    <priority>0.3</priority>
  </url>
</urlset>
```
**Влияние:** Критичное для SEO. Без sitemap снижается скорость и полнота индексации.

### 3. Отсутствует Meta Description на главной странице
**Где:** https://mafclub.biz/ (главная)
**Проблема:** Тег `<meta name="description">` не найден в HTML. Поисковики будут генерировать описание автоматически.
**Как исправить:**
- Добавить в `<head>`:
```html
<meta name="description" content="Мафклуб для лидеров в Москве. Игра в спортивную мафию с профессиональным ведущим. Развивайте софт-скиллы: убеждение, чтение людей, лидерство. Забронируйте место по цене 3500₽ до 20 января.">
```
**Влияние:** Критичное для SEO и конверсии. Meta description влияет на CTR в поисковой выдаче.

### 4. Отсутствует H1 на главной странице
**Где:** https://mafclub.biz/ (главная)
**Проблема:** На странице нет явного тега H1. "Мафклуб для лидеров" должен быть обёрнут в H1.

### 5. Отсутствует Canonical URL

### 6. Отсутствует viewport meta tag
**Где:** https://mafclub.biz/ (главная)
